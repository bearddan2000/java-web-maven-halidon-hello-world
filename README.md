# java-web-maven-halidon-hello-world

## Description
A POC for oracle's halidon microframework.

## Tech stack
- java
- maven
  - helidon
  - netty

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
Available at http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
